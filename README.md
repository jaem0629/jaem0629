
**"Fresh & Clean"** 이라는 철학으로 깔끔하고 읽기 쉬운 코드를 추구하는 소프트웨어 엔지니어 이재민입니다.

생텍쥐페리의 "완벽은 뺄 것이 없을 때 성취된다"는 말을 좋아합니다. 중복을 줄이고, 구조는 단순하게 유지합니다.

단순히 문제를 해결하는 것을 넘어 팀 전체의 생산성을 높이는 방법을 고민합니다. 타입 자동화, 캐싱, 성능 최적화를 통해 개발자와 사용자 경험을 함께 개선하는 것에 보람을 느낍니다.

React, TypeScript를 주력으로 사용하며, GraphQL을 통한 선언적 데이터 페칭에 관심이 많습니다.

여가 시간엔 새로운 기술을 학습하거나, 게임을 하거나, 재즈를 들으며 영감을 얻습니다.

지금은 퍼블릭에이아이에서 프론트엔드 개발자로 일하고 있습니다.
